Getting started

It’s very easy to get started with Fannypack. Just include a link to our css file in your <head> and you’re good to go.

Next: start implementing the html.

Where is feature X or Module Y?

Our goal is to keep Fannypack lightweight so we keep it to the basic modules. If you need something feel free to submit pull requests or report any bugs in our Github repo.
